# .claude/skills/

Installable Cowork skills for this practice. Each skill is a folder with a `SKILL.md` file containing YAML frontmatter (name, description) and the prompt body. Skills fire on natural-language phrases described in the frontmatter; you do not invoke them by name. Add new skills here by creating a new folder with its own `SKILL.md`.
